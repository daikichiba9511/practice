# fast api practice

## 概要

- web applicationを作るのに慣れる為に、FastAPIを使って色々試行錯誤してみる

## 環境構築

### install

- pip

```bash
pip install fastapi uvicorn
```

- conda

```bash
conda install -c conda-forge fastapi uvicorn
```
